# Чек-лист тестирования - Неделя 2

## Регистрация и Вход
- [ ] Можно создать нового пользователя через форму регистрации.
- [ ] Нельзя создать пользователя с уже существующим именем.
- [ ] Можно войти под созданным пользователем.
- [ ] При неверном пароле выводится ошибка.
- [ ] После входа в шапке отображается "Привет, [имя]!".
- [ ] Кнопка "Выход" удаляет сессию и возвращает на главную.

## Услуги
- [ ] Список услуг загружается с бэкенда и отображается на главной.
- [ ] Обычный пользователь не видит ссылку на "Услуги (Админ)".

## Админка (Управление услугами)
- [ ] Первый зарегистрированный пользователь получает роль `admin`.
- [ ] Админ видит ссылку "Услуги (Админ)" в навигации.
- [ ] Админ может добавить новую услугу через форму.
- [ ] Новая услуга сразу появляется в списке на главной.
- [ ] Админ может удалить услугу.
- [ ] Обычный пользователь не может зайти на `/admin/services` (бэкенд должен возвращать 403).
