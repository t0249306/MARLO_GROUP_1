# Чек-лист тестирования - Неделя 3

## Создание записи
- [ ] Обычный пользователь может выбрать услугу и дату/время.
- [ ] Нельзя создать запись на прошедшую дату (ошибка бэкенда).
- [ ] После записи происходит перенаправление на страницу "Мои записи".
- [ ] Неавторизованный пользователь при попытке записи перенаправляется на страницу входа.

## Просмотр и отмена (Пользователь)
- [ ] На странице "Мои записи" отображаются только записи текущего пользователя.
- [ ] Каждая запись содержит название услуги, дату, цену и статус.
- [ ] Кнопка "Отменить запись" удаляет её из списка.
- [ ] Пользователь не может отменить чужую запись через API (проверка `user_id` на бэкенде).

## Просмотр записей (Админ)
- [ ] Админ видит ссылку "Все записи (Админ)".
- [ ] В админке отображаются записи всех пользователей с указанием их имен.
- [ ] Обычный пользователь не имеет доступа к списку всех записей.
