# Чек-лист тестирования - Неделя 1

## Backend
- [ ] Сервер запускается командой `npm run dev` в папке `backend`.
- [ ] Скрипт инициализации БД (`npm run init-db`) создает 3 таблицы: `users`, `services`, `bookings`.
- [ ] Эндпоинт `GET /api/health` возвращает `{ "status": "OK" }`.
- [ ] Эндпоинт `GET /api/services` возвращает список услуг (или пустой массив) из БД.

## Frontend
- [ ] Фронтенд запускается командой `npm run dev` в папке `frontend`.
- [ ] Главная страница отображает заголовок "Система онлайн-записи".
- [ ] Фронтенд успешно делает запрос к бэкенду (в консоли нет ошибок CORS).
- [ ] Если в БД есть услуги, они отображаются на странице.

## База данных
- [ ] Таблица `users` имеет колонки `id`, `username`, `password_hash`, `role`.
- [ ] Таблица `services` имеет колонки `id`, `name`, `description`, `price`.
- [ ] Таблица `bookings` имеет колонки `id`, `user_id`, `service_id`, `booking_date`, `status` и внешние ключи.
